About
This project classifies every US public elementary school (~67,000) by modeled highway noise exposure and tests for disproportionate impact in low-income communities. Noise levels are estimated using a physics-based ISO 9613-2 attenuation model combined with locale-calibrated urban noise floors derived from NCES locale codes.
